Step 1: Import your personal certificates

You can use the following procedures to manually insert your certificates into your Firefox browser. You will have to import your signature and encryption certificates separately.

  1. Open your browser in Firefox.
  2. Select "Tools" from the menu bar and then select "Options". If you are in more recent version of Firefox, just select the Firefox dropdown, "Options" and then "Options" again.
  3. The Firefox Options window appears. Select "Advanced" from the top menu. Select the "Encryption" tab and the "View Certificates" button.
  4. The Certificate Manager window appears. Select the "Your Certificates" tab and the "Import" button.
  5. The File Name to Restore window appears. Choose the certificate you would like to import and click "Open".
  6. The Password Required window appears.
  7. Enter the master password that you created and select "OK". The Password Entry Dialog box appears.
  8. Enter the PKI password you created during the PKI enrollment process. Select "OK".
  9. The successful alert message will appear. Select "OK". Repeat these steps to import the remaining PKI certificate into your Firefox browser.

Step 2: Edit Your Certificate Trust Settings

  1. Open your browser in Firefox.
  2. Select "Tools" from the menu bar and then select "Options".
  3. The Firefox Preferences window appears.
  4. Select "Advanced" from the top menu. Select the Encryption tab and "View Certificates". The Certificate Manager window appears.
  5. Select "Authorities". A list of Certificate Authorities appears. Use the scroll bar to locate the section titled U.S. Government. Select the "IC DIA CA 3" CA under U.S. Government, highlight it, and select "Edit".
  6. The Edit Security Certificate Settings window appears.
  7. Ensure that all three check boxes are checked.
  8. Select "OK". The Edit Security Certificate Settings window closes. The list of Certificate Authorities reappears.
  9. Use the scroll bar to locate the section titled U.S. Government. Select the "IC PKI ROOT 2" CA under U.S. Government, highlight it, and select "Edit".
  10. The Edit Security Certificate Settings window appears.
  11. Ensure that all three check boxes are checked.
  12. Select "OK". The Edit Security Certificate Settings window closes.
